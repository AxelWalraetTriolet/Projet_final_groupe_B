# F1 Strategy Simulator

## Description
Ce projet est développé dans le cadre du cours MGA 802. Il s'agit d'un simulateur d'aide à la décision et d'optimisation de stratégie de course de Formule 1. L'application utilise la télémétrie réelle via `FastF1` et permet de tester des scénarios de course (choix des pneus, arrêts aux stands) face à des modèles de dégradation physiques et stochastiques à travers une interface `Streamlit`.

## Structure du Projet
* `config.yaml` : Fichier de configuration des constantes et paramètres.
* `app.py` : Point d'entrée de l'application Streamlit.
* `src/` : Dossier contenant le code source de la bibliothèque (Classes POO).
